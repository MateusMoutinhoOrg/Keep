# Contribution Rules

Rules to follow when contributing to this project. All documentation must also comply with [DocumentationStandards.md](./DocumentationStandards.md).

---

## File Changes

Before creating, deleting, or renaming any file or directory, read [STRUCTURE.md](./STRUCTURE.md) and check whether the change affects the project structure. If it does, update [STRUCTURE.md](./STRUCTURE.md) in the same commit.

---

## Documentation Changes

When you create, delete, or rename a `.md` file, update the Internal Docs section of [README.md](../../README.md).

---

## Sample Changes

When you create, delete, or rename a sample (any file inside [examples/](../../examples)), update the Samples section of [README.md](../../README.md).
